Ducky Script Library
=====================

A library of ducky scripts for practical HACKING uses. Licensed under the MIT License.

Uses
=====

There's a GNU C Library, so why not a Ducky Script Library? You can use these scripts under the MIT license, so use them to your hearts content! There is a file coming up called "examples.md" that will have examples for uses.
